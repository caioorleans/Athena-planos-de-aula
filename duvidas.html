DÚVIDAS FREQUENTES

COMO POSSO FAZER O DOWNLOAD COMPLETO DO REPOSITÓRIO ATHENA?


  1. Clique em “Início” e role a tela até chegar na opção de fazer download do repositório. Clique em “Download”.




  2. Clique com o botão direito no arquivo baixado e selecione a opção “Extrair aqui” para extrair o Repositório Athena.




  3. Após a extração, surgirá uma pasta com todo o repositório. Abra a pasta e prossiga para o próximo passo.




  4. O arquivo de nome “index” possibilita o acesso ao repositório de forma off-line.




  5. A pasta com o nome “recursos” contém todos os recursos do repositório. Abra e siga para o próximo passo.




  6. Na pasta “recurso” estão todos os RED para computador. Basta abrir, selecionar a disciplina e o ano desejado para encontrar todos os RED da categoria escolhida.




QUAL É O TAMANHO RECOMENDADO DE UM PEN DRIVE PARA O DOWNLOAD COMPLETO DO REPOSITÓRIO ATHENA?


Para o download do Repositório Athena, você precisará de um pen drive com capacidade de, no mínimo, 4gB. Esse é o tamanho necessário para o armazenamento de todo o conteúdo do repositório.

COMO EXTRAIR OS ARQUIVOS COM OS RED PARA COMPUTADOR APÓS O DOWNLOAD?


    Os RED para computador disponíveis no Repositório Athena são compactados em um formato de arquivo que facilita o seu transporte. Pode-se dizer que o recurso é colocado dentro de um pacote e, para abrir este pacote, é necessário utilizar um programa de extração. Sugerimos utilizar o WinRAR ou PeaZip.

    Os próximos passos mostram como fazer o download desses programas. Escolha somente um deles.

    WinRAR: Clique aqui para acessar o site do programa. Confira se a linguagem está para “Português brasileiro” e, em seguida, clique em “Continuar para baixar WinRAR”.




    PeaZip: Clique aqui para acessar o site do programa. Na imagem, duas opções de download são mostradas. Selecione a opção adequada para o sistema do seu computador.




    A seguir, saiba como extrair os RED.

  1. Para extrair o RED, clique com o botão direito no arquivo baixado e selecione a opção “Extrair aqui”.




  2. Abra o RED, utilizando o Adobe Flash Projector.



  3. Pronto! Agora você já possui o RED selecionado em seu computador.



COMO POSSO FAZER O DOWNLOAD DOS RECURSOS PARA COMPUTADOR?


  1. No menu superior, selecione “Para Computador” na opção “Recursos”.




  2. Na barra de busca, selecione uma disciplina, o ano escolar e, se desejar, um descritor. Clique em “Buscar” para filtrar os recursos.




  3. Escolha um dos RED disponíveis. (Para esse passo a passo, utilizaremos, como exemplo, o RED “Brincando com as vogais”).




  4. Clique em “Download” para baixar o RED.




  5. Clique com o botão direito no arquivo baixado e selecione a opção “Extrair aqui” para extrair o RED.




  6. Após isso, o recurso estará no formato .swf e poderá ser executado com o auxílio do programa Adobe Flash Projector. Caso ainda não possua o programa em seu computador, clique aqui para acessar o rutorial de como baixá-lo para, em seguida, executar o RED.




O ATHENA PRECISA DE ALGUM PROGRAMA ESPECÍFICO PARA A EXECUÇÃO DOS RED PARA COMPUTADOR?


É necessário a instalação do programa "Adobe Flash Player Projector", atente para versão compatível com o sistema operacional do seu computador (Windows, Mac ou Linux).
Abaixo, segue um passo a passo de como baixar e configurar o programa para utilização dos recursos em "Flash".

  1. Acesse o site Adobe Flash Player, role para baixo até encontrar a opção de download do Adobe Flash Player Projector Compatível com seu sistema operacional.




  2. Selecione o local onde o arquivo será salvo e clique em salvar. Caso essa janela não apareça, significa que o arquivo foi salvo no local padrão de download do seu computador.




  3. Para configurar, usaremos, como exemplo, o RED "A História da Páscoa". Clique aqui para baixá-lo.

  4. Extraia o RED, clicando com o botão direito do arquivo baixado e selecione "Extrair aqui..."




  5. Após esses passos, o RED estará na sua pasta. Agora, precisamos configurá-lo para abrir com o "Adobe Flash Player" que baixamos anteriormente.




  6. Clique com o botão direito no RED, mova o mouse para a opção "Abrir com" e selecione a opção "Escolher outro aplicativo".




  7. Certifique-se de que a caixa de "Sempre usar este aplicativo para abrir arquivos .swf" esteja marcada e clique em "Mais aplicativos".




  8. Role a barra lateral para baixo e selecione a opção "Procurar outro aplicativo neste PC".




  9. Na janela que abriu, vá até a pasta do local onde você salvou o programa "Adobe Flash Player" (por padrão, está com o nome de "flashplayer_32_sa"), selecione-o e clique em "Abrir".




  10. Agora todos os recursos no formato .swf funcionarão em seu computador! Esse procedimento só é necessário ser feito uma vez. Após feito, para abrir um RED nesse mesmo formato, basta clicar duas vezes no próprio recurso.




POR QUE OS LINKS DOS APLICATIVOS EDUCACIONAIS PODEM NÃO FUNCIONAR?


Por estarem disponíveis na Play Store e por sofrerem constantes atualizações, os links dos aplicativos disponibilizados no Repositório Athena podem, eventualmente, aparecer inativos. Buscando manter o repositório sempre atualizado, os links são constantemente revisados, de forma a manter no repositório somente aplicativos disponíveis na Play Store.
Caso o aplicativo selecionado apresente algum erro, informe nossa equipe: athena@virtual.ufc.br.
